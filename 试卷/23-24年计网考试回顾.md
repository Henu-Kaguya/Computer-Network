# 2023-2024学年 计算机网络考试回顾（①卷）

**基本信息：**

* **时间：** 2023年12月12日（星期二）
* **分值分布：**
* 选择题：15题  2分 = 30分
* 填空题：10题  2分 = 20分
* 简答题：5题  6分 = 30分
* 综合分析题：20分

## 一、 综合分析题 (20分)

1. **路由与网关配置**

* **场景图示：** [H1] -- [交换机] -- [路由器] -- [H2/H3]
* **问题 (1)：** 给定 H1 的默认网关；(推测) 当 H1 给 H3 通信时，路由器提供什么服务？
* *(注：考察路由器的转发功能及网关概念)*

2. **FTP 协议**

* **问题：** FTP 默认端口号是多少？建立什么连接？
* *(注：通常考察控制连接端口 21 和数据连接端口 20)*

3. **IP 数据报分片**

* **问题：** 给定 1500 字节的数据报进行分片，每片 MTU 最多 700B。求每片的数据量（长度）和片偏移量（Fragment Offset）。
* *(注：需扣除 IP 首部 20B 进行计算)*

4. **CIDR 子网划分**

* **问题：** 给 M 个分区，3个区各包括 100台、45台、45台主机，写出 CIDR 地址块分配方案。
* *(注：考察 VLSM 可变长子网掩码划分)*

5. **华为路由器命令配置**

* **问题：** 用指令给接口赋 IP 地址。
* **代码填空：**

```bash
<Huawei> system-view
[Huawei] interface g 0/0/?  (进入接口)
[Huawei-GigabitEthernet0/0/?] ip address _________________ (配置IP)

```

## 二、 简答题 (5题 × 6分)

1. **性能指标计算**

* **问题：** 求发送的数据帧长。图示给出距离 1km。已知数据率和传播速率。
* *(注：可能考察“发送时延=传播时延”时的帧长，或 CSMA/CD 的最短帧长)*

2. **CDMA (码分多址)**

* **问题：** 求 A 向 C 发送了什么数据。题目给出了 A、B、C 的码片序列。
* *(注：考察 CDMA 的规格化内积运算)*

3. **CRC (循环冗余校验)**

* **问题：** 冗余检验算法，给定生成多项式 ，求发送的原数据（或求FCS/余数）。

4. **路由算法**

* **问题：** 更新路由表，求下一跳。
* *(注：通常考察距离向量算法 RIP 的路由表更新规则)*

5. **TCP 拥塞控制**

* **问题：** 拥塞控制的 4 种算法（慢开始、拥塞避免、快重传、快恢复）。
* **计算：** 已知一次往返时延 (RTT) 为 10ms，数据报最大 2KB，问多久能达到 24KB？

## 三、 选择题 (15题 × 2分)

*(笔记中列出了一些考点关键词)*

1. **应用层协议与传输层关系：**

* 哪些没用 TCP？(选项包含：DHCP, SMTP, TELNET, FTP)。
* *(答案提示：DHCP 使用 UDP)*

2. **物理介质：**

* 100BASE-T 用的是什么介质？(选项：UTP, STP, 光纤, 同轴电缆)。
* *(答案提示：UTP 非屏蔽双绞线)*

3. **体系结构：**

* 哪层实现端到端通信？
* *(答案提示：传输层)*

4. **帧长计算：**

* 求帧长，结果为 1250B。

5. **路由查找步骤：**

* 特定主机路由 > 网络路由 > 默认路由。

6. **TCP 窗口控制：**

* 拥塞窗口 (cwnd) 和 接收窗口 (rwnd)，发送窗口取值 = 。

7. **邮件协议：**

* 发送邮件的协议：SMTP, HTTP。
* *(注：通常发邮件用 SMTP，Web 邮箱用 HTTP，收邮件用 POP3/IMAP)*

8. **UDP 特征：**

* 面向报文、无连接、不可靠...

9. **DNS 解析：**

* 域名服务器：主机和本地域名服务器。发送的请求：一条（递归查询） vs 多条（迭代查询）。

10. **填充相关：**

* *(笔记模糊)* 提到“填充后的比特数据”，可能指以太网帧的最短长度填充。

## 四、 填空题 (10题 × 2分)

1. **物理层接口特性：**

* 四大特性：机械特性、电气特性、功能特性、规程(过程)特性。

2. **数据链路层信道：**

* 两个信道类型：点对点信道、广播信道。

3. **TCP 序列号：**

* 题目涉及 30 和 100，问第二次报文的**确认序号**：100。
* *(注：确认号通常是期待收到的下一个字节的序号)*

4. **万维网 (WWW)：**

* WWW 用 **URL** 标识互联网的文档。

5. **PING 命令：**

* PING 用的 **ICMP** 协议。

6. **交换机原理：**

* 根据帧的 **目的 MAC 地址** 查表转发。

7. **计算机网络的功能：**

* 计网提供的功能：**数据通信(连通性)** 和 **资源共享**。

8. **UDP 首部与端口：**

* 画了一个 UDP 首部图。
* 求源端口号：**69**。
* *(注：69 是 TFTP 的端口，TFTP 基于 UDP)*
